# Data Folder

This folder contains information about the datasets used in this project.

The raw and processed datasets are not uploaded to this repository due to file size and reproducibility reasons.

---

## Source Dataset

Dataset used:

**Brazilian E-Commerce Public Dataset by Olist**

Source: Kaggle
Kaggle dataset package: `olistbr/brazilian-ecommerce`

---

## Folder Structure

```text
data/
├── raw/
├── processed/
└── README.md
```

| Folder       | Description                                                  |
| ------------ | ------------------------------------------------------------ |
| `raw/`       | Original CSV files downloaded from Kaggle                    |
| `processed/` | Cleaned and transformed datasets generated by Python scripts |

---

## How to Reproduce the Data

To reproduce this project, download the dataset from Kaggle and place all original CSV files inside:

```text
data/raw/
```

---

## Manual Download Instructions

1. Open Kaggle.
2. Search for **Brazilian E-Commerce Public Dataset by Olist**.
3. Download the dataset ZIP file.
4. Extract the ZIP file.
5. Move all `.csv` files into:

```text
olist-ecommerce-bi-analytics/data/raw/
```

---

## Kaggle CLI Download

If you use Kaggle CLI, run this command from the project root folder:

```bash
kaggle datasets download -d olistbr/brazilian-ecommerce -p data/raw --unzip
```
